## Requirements

 - `git`
 - `cmake`
 - `ninja`
 - `clang`
 - `Vulkan SDK`
 - `msvc`
 - `Windows SDK`

## Build

 - Run `build-all.bat` or use `Ctrl + Shift + B` shortcut in vscode.

## Run

 - Run `F5` shortcut in vscode.
